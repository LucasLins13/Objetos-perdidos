# Achados & Perdidos - Starter

Projeto starter com Vue 3 + Vite + Tailwind + Firebase (Firestore + Storage).

Instruções:
1. Copie `.env.example` para `.env` e preencha com suas credenciais Firebase.
2. `npm install`
3. `npm run dev`

Funcionalidades incluídas:
- Listagem de itens
- Upload de imagem (Storage)
- Marcar item como recuperado
- Filtragem por texto / status
- Link para conversar com Adriana via WhatsApp
